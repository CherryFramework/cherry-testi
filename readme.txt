A testimonials management plugin for WordPress.

== Description ==

This plugin is designed to help you add testimonials and reviews to the site and customize their look in accordance with the design of the website. Testimonials can also be added via shortcodes to any page or post:

`[tm_testimonials]`

== Installation ==

1. Upload Cherry Testi folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Testimonials->Settings screen to configure the plugin
